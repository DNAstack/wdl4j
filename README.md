## Getting Started

### Building


1. Clone this repository

```shell script
git clone http://github.com/dnastack/wdl4j.git .
cd wdl4j
```

2. Build the source code using the packaged maven wrapper

```shell script
./mvnw clean compile
```
3. Run the tests

```shell script
./mvwn test
```
